### 🛠️ Teknik Altyapı ve Mimari
GöreviO, ölçeklenebilir ve modüler bir yapıda, **Feature-Based** mimari prensipleriyle geliştirilmiştir.

- **Frontend:** SwiftUI, Combine, URLSession (REST API Integration)
- **Backend:** Node.js, Next.js API Routes, Vercel
- **Database:** MongoDB Atlas (Cloud Database)
- **Güvenlik:** Role-Based Access Control (Admin & Personnel)

### 🚀 Uygulama Özellikleri
- **Admin Paneli:** Saha personeli yönetimi, anlık görev atama ve operasyonel takip.
- **Personel Arayüzü:** Görev listeleme, detaylı iş inceleme ve anlık durum güncellemeleri.
- **Veri Senkronizasyonu:** Bulut tabanlı veritabanı sayesinde tüm cihazlar arasında gerçek zamanlı veri akışı.
